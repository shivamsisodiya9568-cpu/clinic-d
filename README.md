# GS Clinic - Responsive Healthcare Website

This project is a premium, modern, and fully responsive website designed for "GS Clinic" located in Agra, Uttar Pradesh, India. It aims to provide an exceptional online presence, making it easy for patients to access information and book appointments.

## Features

*   **Responsive Design**: Optimized for Desktop PC, Laptop, Tablet, and Mobile Phone using Flexbox, CSS Grid, media queries, fluid typography, and responsive images.
*   **Modern Medical Theme**: Professional UI with clean icons, attractive buttons, glassmorphism cards, gradients, and smooth animations.
*   **Animated Hero Section**: Engaging landing area to capture visitor attention.
*   **Key Clinic Sections**:
    *   About Clinic: Detailed information about GS Clinic.
    *   Doctor Profiles: Showcase of expert medical professionals.
    *   Services: Comprehensive list of medical services offered.
    *   Appointment Booking Form: User-friendly form for scheduling visits.
    *   Patient Reviews: Testimonials from satisfied patients.
    *   Gallery: Visual tour of clinic facilities.
    *   Health Tips: Informative articles for well-being.
    *   FAQ Accordion: Answers to common patient queries.
    *   Contact Information & Google Map: Easy access to clinic location and details.
*   **Interactive Elements**:
    *   Sticky Navbar: Stays visible for easy navigation.
    *   Mobile Menu: Collapsible navigation for smaller screens.
    *   Loading Animation: Smooth pre-loading experience.
    *   Smooth Scrolling: Seamless navigation between sections.
    *   Back to Top Button: Quick scroll to the top of the page.
    *   Floating WhatsApp & Call Now Buttons: Instant contact options.
*   **AI Clinic Assistant Chatbot**: A floating chatbot named "GS Clinic AI Assistant" that can answer questions about timings, location, services, appointments, doctor availability, and emergencies using JavaScript.
*   **Touch-Friendly Buttons**: Enhanced usability on mobile devices.
