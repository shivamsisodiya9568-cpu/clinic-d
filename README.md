# GS Clinic Website

This project is a modern, responsive, and professionally designed website for "GS Clinic" located in Agra, India. It's built with HTML, CSS, and JavaScript, focusing on a premium user experience, smooth animations, and essential clinic information.

## Features

*   **Fully Responsive Design:** Adapts seamlessly to desktops, laptops, tablets, and mobile phones.
*   **Modern Medical Theme:** Clean design with smooth animations, glassmorphism cards, gradients, and professional color palettes.
*   **Animated Hero Section:** Engaging video background with dynamic text and call-to-action buttons.
*   **Comprehensive Sections:** Includes About Us, Doctor Profiles, Services, Appointment Booking, Patient Reviews, Gallery, Health Tips, FAQ, Location Map, and Footer.
*   **Sticky Navbar & Mobile Menu:** Intuitive navigation with a sticky header and a functional hamburger menu for mobile.
*   **Interactive Elements:** Smooth scrolling, back-to-top button, accordion FAQs, and responsive image handling.
*   **Floating AI Clinic Assistant:** A chatbot integrated via JavaScript to answer common questions about timing, location, services, appointments, and doctor availability.
*   **Call & WhatsApp Buttons:** Prominent floating buttons for immediate contact.
*   **Preloader Animation:** A smooth animation while the page content loads.
*   **Touch-Friendly UI:** Optimized for easy interaction on touch devices.
